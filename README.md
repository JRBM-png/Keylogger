# Keylogger
This repository holds a basic key logger for reading inputs from a keyboard. 
## Objective 
The main objective was to create a python file that could record inputs from a keyboard and record them in a log.txt file. Additonally my aim was by the end of the project was to understand what exactly a keylogger is, what it does and why it can be a threat to security. 
## Skills learnt
- Python coding
- Execption handing for special/non-character keys
- Python librarys
## Tools Used
- VS code
## Steps
### Prerequsities 
Install the python libirary pynput Via this command:

pip install pynput
### Run the file
Run the file in your IDE and when you type it will automatically create a log.txt file, you can open it to view everything you've typed.
